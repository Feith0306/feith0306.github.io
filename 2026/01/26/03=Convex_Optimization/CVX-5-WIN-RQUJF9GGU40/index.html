凸优化学习系列 - 5. 次梯度法

目标：读完这篇博客，应当能

  • 理解次梯度与次微分的必要性并会算常见例子；
  • 学会如何用次梯度法求解凸优化问题；
  • 知道为何次梯度法收敛慢、收敛到哪、需要什么前提条件；

1. 问题描述

1.1 目标函数可微

对于凸优化问题： \[\begin{align} \label{eq:unconstrained} \min_{x} f(x), \end{align}\] 根据凸优化学习系列 - 4. 梯度下降法中的讨论，如果 \(f\) 可微，则可按照梯度下降更新 \[\begin{align} \label{eq:gd-update} x_{k+1}=x_{k}-t\nabla f(x_{k}). \end{align}\] 来求解问题 \(\eqref{eq:unconstrained}\)。

但总有些函数 在某些点不可微，也就是说在某些点梯度不存在。

梯度不存在，怎么进行“梯度下降”更新？怎么求解 \(\eqref{eq:unconstrained}\)？

1.2 目标函数不可微

注意到，梯度本质上是可微函数的一阶信息：对于可微函数 \(f\)，在点 \(x\) 处，梯度 \(\nabla f(x)\) 满足 \[\begin{align} \label{eq:1st-order} f(y)\ge f(x)+\nabla f(x)^\top (y-x),\quad \forall x,y. \end{align}\]

上式表明，可以用梯度刻画可微函数 \(f\) 在某点的下界。实际上，上式为可微凸函数的一阶凸性条件（first-order condition），是一个充分条件（具体证明见博客）。

对于不可微的凸函数 \(f\)，我们希望能找到一种推广，使得类似 \(\eqref{eq:1st-order}\) 的不等式依然成立，从而可以用来刻画不可微凸函数在某点的一阶信息。

2. 次梯度与次微分

定义一：次梯度（subgradient）

设 \(f:\mathbb{R}^n\to\mathbb{R}\) 为凸函数。如果对任意 \(y\in\mathbb{R}^n\) 有 \[\begin{align} \label{eq:subgradient} f(y)\ge f(x)+g^\top(y-x). \end{align}\] 则称向量 \(g\in\mathbb{R}^n\) 称为 \(f\) 在点 \(x\) 的一个次梯度。

不等式 \(\eqref{eq:subgradient}\) 与 \(\eqref{eq:1st-order}\) 形式上完全相同，只不过把梯度 \(\nabla f(x)\) 换成了次梯度 \(g\)。因此，次梯度可以看作是对不可微凸函数在某点的一阶信息的推广。

定义二：次微分（subdifferential）

所有次梯度的集合称为次微分： \[\begin{align} \partial f(x)=\{g:\ f(y)\ge f(x)+g^\top(y-x),\ \forall y\}. \end{align}\]

重要性质：

  • 非空性：对于任意凸函数 \(f\) 和任意点 \(x\)，次微分 \(\partial f(x)\) 非空（至少包含一个次梯度）。
  • 有界性：如果 \(f\) 在某点 \(x\) 附近 Lipschitz 连续，则 \(\partial f(x)\) 有界（元素有界）。
  • 一致性：若 \(f\) 在 \(x\) 可微，则 \(\partial f(x)\) 退化为唯一的梯度，即 \(\partial f(x)=\{\nabla f(x)\}\)。
  • 凸性：次微分 \(\partial f(x)\) 是一个凸集（次梯度的任意凸组合仍是次梯度）。
  • 闭性：次微分 \(\partial f(x)\) 是一个闭集。

3. 次梯度法

3.1 最优性条件

对于可微凸函数，最优性条件为： \[\begin{align} f(x^\star) = \min_x f(x) \Leftrightarrow \nabla f(x^\star)=0. \end{align}\] 对于不可微凸函数，最优性条件推广为： \[\begin{align} f(x^\star) = \min_x f(x) \Leftrightarrow 0 \in \partial f(x^\star). \end{align}\] 也就是说，若 \(x^\star\) 最优，则 \(f\) 在 \(x^\star\) 处至少有一个次梯度为零向量。

问题一：为什么“次梯度为零”对应最优解？

解答：由次梯度定义 \(\eqref{eq:subgradient}\) 可知，若 \(0\in\partial f(x^\star)\)，则对任意 \(y\) 有 \[\begin{align} f(y)\ge f(x^\star)+0^\top(y-x^\star)=f(x^\star), \end{align}\] 即 \(x^\star\) 是最优解。

3.2 次梯度法的构造

在 凸优化学习系列 - 4. 梯度下降法 中，我们讨论了梯度下降法的构造思路：基于目标函数的梯度在当前点构造目标函数的二次函数上界，并通过最小化该上界来定义更新。

现在 \(f\) 不可微，如何构造上界？首先，基于次梯度的定义 \(\eqref{eq:subgradient}\)，可知对任意 \(g^{(k)}\in\partial f(x^{(k)})\)，都有 \[\begin{align} f(x) \ge f(x_k) + (g_k)^\top (x - x_k),\quad \forall x. \end{align}\] 上式给出了 \(f(x)\) 在 \(x_k\) 处的一个线性下界。为了得到 \(f(x)\) 上界的（二次）估计，我们可以在该线性下界上再加一个二次正则项，从而得到： \[\begin{align} \label{eq:subgradient-upper-bound} f(x) \le \hat f(x;x_k) = \underbrace{f(x_k) + g_k^\top (x - x_k)}_{\text{线性下界：仿射模型}} + \underbrace{\frac{1}{2t_k}\|x - x_k\|_2^2}_{\text{二次正则项：信赖域}}, \quad \forall x. \end{align}\]

因此，基于 \(\eqref{eq:subgradient-upper-bound}\) 中构造出的近似上界，我们可以将问题 \(\eqref{eq:unconstrained}\) 转化为一系列子问题： \[\begin{align} \label{eq:subgradient-subproblem-1} \min_x \hat f(x;x_k) = \min_x \left( f(x_k) + g_k^\top (x - x_k) + \frac{1}{2t_k}\|x - x_k\|_2^2 \right). \end{align}\] 则下一个迭代点 \(x_{k+1}\) 为该子问题的解： \[\begin{align} \label{eq:subgradient-subproblem-2} x_{k+1} = \arg\min_x \hat f(x;x_k) = \arg\min_x \left( f(x_k) + g_k^\top (x - x_k) + \frac{1}{2t_k}\|x - x_k\|_2^2 \right). \end{align}\]

因此，次梯度法的更新规则为： \[\begin{align} \label{eq:subgradient-update} x_{k+1} = x_k - t_k g_k,\quad g_k \in \partial f(x_k). \end{align}\]

问题二： 这里的参数 \(t_k\) 需要满足什么条件？为什么有角标？

问题三： 次梯度 \(g_k\) 属于集合此微分 \(\partial f(x_k)\)，但该集合通常包含多个次梯度（多个更新方向），到底选哪个？

注意：在合适的初值与步长下，梯度下降法能保证目标函数值单调下降，具体讨论见博客。但次梯度法不保证每步下降，因为不可微点处不一定下降，例如 \(f(x) = |x|\)。

因此，次梯度法常用一个更稳健的输出策略：记录迭代过程中目标函数值最小的点（best iterate），即 \[\begin{align} x_k^{\mathrm{best}} = \arg\min_{i=0,\dots,k} f(x_i). \end{align}\]

注意：

  • “best iterate” 不是算法的更新规则（不会再用 “best iterate” 继续更新），只是输出策略；
  • 角标 \(i\) 只是一个“遍历索引”（没有嵌套子循环）。

3.3 步长选择

3.3.1 固定步长（fixed step）

定理一（固定步长的更新性质）：设 \(f\) 为 Lipschitz 连续的凸函数， \[\begin{align} \label{eq:lipschitz} \exists G>0,\quad |f(x)-f(y)|\le G\|x-y\|_2,\quad \forall x,y, \end{align}\] 则次梯度法 \(\eqref{eq:subgradient-update}\) 在固定步长 \(t_k=t\), \(\forall k\) 下满足： \[\begin{align} \lim_{k\to\infty} f(x_k^{\mathrm{best}}) \le f^\star + \frac{G^2 t}{2}, \end{align}\]

证明：

我们已知（由次梯度不等式/凸性一阶条件） \(g^{\top}_{k-1}(x_{k-1}-x^{\star}) \ge f(x_{k-1})-f^{\star}\)。 于是有 \[\begin{align} \begin{split} \|x_{k}-x^{\star}\|_{2}^{2} &= \|x_{k-1}-t_{k-1}\,g_{k-1}-x^{\star}\|_{2}^{2} \\ &= \|x_{k-1}-x^{\star}\|_{2}^{2}-2t_{k-1}g_{k-1}^{\top}(x_{k-1}-x^{\star})+t_{k-1}^{2}\|g_{k-1}\|_{2}^{2} \\ &\le \|x_{k-1}-x^{\star}\|_{2}^{2}-2t_{k-1}\big(f(x_{k-1})-f(x^{\star})\big)+t_{k-1}^{2}\|g_{k-1}\|_{2}^{2}. \end{split} \end{align}\]

对上述不等式迭代（累加）得到 \[\begin{align} \|x_{k}-x^{\star}\|_{2}^{2} \le \|x^{0}-x^{\star}\|_{2}^{2} -2\sum_{i=1}^{k}t_{i-1}\big(f(x_{i-1})-f(x^{\star})\big) +t_{i-1}^{2}\sum_{i=1}^{k}\|g_{i-1}\|_{2}^{2}. \end{align}\]

注意：\(f\) 为 \(G\)-Lipschitz 等价于对任意 \(i\) 均有 \(\|g_{i}\|_{2}\le G\)， 从而 \(\|g_{i}\|_{2}^{2}\le G^{2}\)；并且 \(\|x_{k}-x^{\star}\|_{2}^{2}\ge 0\)。 令 \(R=\|x_{0}-x^{\star}\|_{2}\)，则由上式可得 \[\begin{align} 0 \le R^{2} -2\sum_{i=1}^{k} t_{i-1}\,\big(f(x_{i-1})-f(x^{\star})\big) +G^{2}\sum_{i=1}^{k} t_{i-1}^{2}. \end{align}\]

由此得到一个基本不等式，我们可以据此直接读出收敛结果。令 \(x^{\mathrm{best}}_{k}\) 表示 best iterate，则有 \[\begin{align} \label{eq:basic-inequality} f(x^{\mathrm{best}}_{k})-f(x^{\star}) \le \frac{R^{2}+G^{2}\sum_{i=1}^{k} t_{i-1}^{2}} {2\sum_{i=1}^{k} t_{i-1}}. \end{align}\]

因此，当采用固定步长 \(t_{i}=t\) 时， \[\begin{align} f(x^{\mathrm{best}}_{k})-f(x^{\star}) \le \frac{R^{2}}{2kt}+\frac{G^{2}t}{2} \ \xrightarrow[k\to\infty]{}\ \frac{G^{2}t}{2}. \end{align}\]

固定步长只能保证收敛到一个由步长 \(t\) 刻画的邻域 ，如果 \(t\) 不趋零，就无法逼近最优值到任意精度。

3.3.2 递减步长（diminishing step）

选择步长序列 \(\{t_k\}\) 满足 \[\begin{align} \label{eq:diminishing-step} \sum_{k=1}^\infty t_k^2<\infty,\qquad \sum_{k=1}^\infty t_k=\infty. \end{align}\]

上式称为Robbins–Monro 条件，典型例子：\(t_k=\frac{1}{k^a}\)，其中 \(\frac{1}{2} < a \le 1\)。

定理二（递减步长的更新性质）：设 \(f\) 为 Lipschitz 连续的凸函数 \(\eqref{eq:lipschitz}\)， 则次梯度法 \(\eqref{eq:subgradient-update}\) 在递减步长 \(\eqref{eq:diminishing-step}\) 下满足： \[\begin{align} \lim_{k\to\infty} f(x_k^{\mathrm{best}}) = f^\star , \end{align}\]

证明：
在定理一 的证明基础上，采用递减步长 \(\eqref{eq:diminishing-step}\)，由 \(\eqref{eq:basic-inequality}\) 可知： \[\begin{align} f(x^{\mathrm{best}}_{k})-f(x^{\star}) \le \frac{R^{2}+G^{2}\sum_{i=1}^{k} t_{i-1}^{2}} {2\sum_{i=1}^{k} t_{i-1}}. \end{align}\] 由于 \(\sum_{i=1}^{\infty} t_{i-1}^{2} < \infty\)，则分子有界；而由于 \(\sum_{i=1}^{\infty} t_{i-1} = \infty\)，则分母趋于无穷大。因此，右侧整体趋于零，即 \[\begin{align} \lim_{k\to\infty} f(x_k^{\mathrm{best}}) = f^\star . \end{align}\]

满足 \(\eqref{eq:diminishing-step}\) 的递减步长能保证目标函数 \(f(x)\) 收敛到最小值。

3.3.3 Polyak 步长（知道最优值 \(f^\star\) 时）

如果最优值 \(f^\star\)已知，Polyak 步长定义为： \[\begin{align} \label{eq:polyak-step} t_k=\frac{f(x^{(k-1)})-f^\star}{\|g^{(k-1)}\|_2^2}. \end{align}\]

定理三（Polyak 步长的更新性质）：设 \(f\) 为 Lipschitz 连续的凸函数 \(\eqref{eq:lipschitz}\)， 则次梯度法 \(\eqref{eq:subgradient-update}\) 在 Polyak 步长 \(\eqref{eq:polyak-step}\) 下满足： \[\begin{align} \lim_{k\to\infty} f(x_k^{\mathrm{best}}) = f^\star , \end{align}\]

4. 收敛性分析

定理三（Polyak 步长的更新性质）：设 \(f\) 为 Lipschitz 连续的凸函数 \(\eqref{eq:lipschitz}\)， 则次梯度法 \(\eqref{eq:subgradient-update}\) 满足： \[\begin{align} f(x^{(k)}_{\mathrm{best}})-f^\star \ge \frac{RG}{2(1 + \sqrt{k+1})}, \end{align}\] 其中 \(R=\|x^{(0)}-x^\star\|_2\)。

5. 实操：次梯度法求解凸优化的方法实现

5.1 伪代码

算法 1: 次梯度法（无约束；返回 best iterate）

输入：初值 \(x_0\)；步长规则 \(\{t_k\}\)（fixed / diminishing / Polyak）；最大迭代次数 \(K\)

输出：\(x_{\mathrm{best}}\)，\(f_{\mathrm{best}}\)

\[ \begin{array}{l} \textbf{Algorithm 1: Unconstrained Subgradient Method}\\ \textbf{Input:}\; x_0,\; \{t_k\},\; K \qquad 1:\; x \leftarrow x_0\\ 2:\; x_{\mathrm{best}} \leftarrow x_0\\ 3:\; f_{\mathrm{best}} \leftarrow f(x_0)\\ 4:\; \textbf{for } k = 1,2,\ldots,K \textbf{ do}\\ 5:\; \qquad \text{choose any } g \in \partial f(x)\\ 6:\; \qquad t \leftarrow \text{stepsize}(k,x,g)\quad (\text{fixed/diminishing/Polyak})\\ 7:\; \qquad x \leftarrow x - t\, g\\ 8:\; \qquad \textbf{if } f(x) < f_{\mathrm{best}} \\ 9:\; \qquad\qquad x_{\mathrm{best}} \leftarrow x\\ 10: \qquad\qquad f_{\mathrm{best}} \leftarrow f(x)\\ 11:\; \qquad \textbf{end if}\\ 12:\; \textbf{end for}\\ \textbf{Output:}\; (x_{\mathrm{best}}, f_{\mathrm{best}})\\[2pt] \end{array} \]
