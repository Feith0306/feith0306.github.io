Projected Self-Triggered Reinforcement Learning for Differential Games

Methodological Framework

Overview

This research line develops reinforcement learning methods for continuous-time differential games under intermittent execution. The key motivation is that many learning-based control algorithms assume continuous monitoring or frequent sampling, which is expensive for networked, embedded, or resource-limited cyber-physical systems.

The work focuses on zero-sum robust control and multi-player nonzero-sum games. It combines self-triggered mechanisms, projection operators, Actor–Critic learning, disturbance learning, fuzzy approximation, and experience replay to obtain stable policies with reduced sampling burden.

Research Questions

  • How can continuous monitoring be avoided? The controller should determine the next update instant from currently available information rather than checking triggering conditions continuously.
  • How can adaptive weight drift be prevented? Projection operators are introduced to keep Actor and disturbance-network weights inside prescribed compact sets during online learning.
  • How can game-theoretic policies be learned under intermittent updates? The learning algorithm must approximate Nash or saddle-point policies even when control updates occur only at selected instants.
  • How can Zeno behavior be excluded? A positive lower bound on inter-execution intervals is required so that infinitely many updates cannot occur in finite time.

Technical Core

  • Self-triggered execution: the next triggering instant is computed from a prediction-based condition, reducing redundant communication and sampling.
  • Projected Actor–Critic learning: projection operators are embedded into adaptive laws to guarantee bounded neural-network weights and improve learning reliability.
  • Actor–Critic–Disturbance structure: the Actor learns the control policy, the Critic approximates the value function, and the disturbance network learns the worst-case input in zero-sum games.
  • Fuzzy reinforcement learning: fuzzy logic systems approximate unknown nonlinear game dynamics and support policy learning for multi-player nonzero-sum games.
  • Experience replay: finite historical data are reused in the Bellman-error minimization process to improve data efficiency without relying solely on current samples.

Paper-Based Contributions

  • Self-triggered H∞ control: a projected Actor–Critic–Disturbance framework is developed for nonlinear quadratic zero-sum games, addressing continuous monitoring and adaptive-weight boundedness simultaneously.
  • Game-theoretic fuzzy RL: self-triggered fuzzy reinforcement learning is extended to multi-player nonzero-sum differential games, where each player learns an approximate Nash policy under intermittent execution.
  • Sampling design for linear systems: self-triggered sampling is studied for continuous-time linear dynamical systems as a foundational mechanism for resource-efficient control.
  • Communication-efficient differential games: linear quadratic zero-sum and nonzero-sum games are used to examine how triggering frequency, learning accuracy, and closed-loop performance trade off.

Representative Outcomes

  • Continuous state monitoring is removed from the execution mechanism.
  • The adaptive weights of learning networks remain bounded through projection-based updates.
  • The closed-loop system and learning errors are proven to be uniformly ultimately bounded under stated assumptions.
  • Zeno behavior is excluded by establishing a positive lower bound on inter-triggering intervals.
  • Sampling and communication loads are reduced while preserving approximate Nash or H∞ control performance.

Application Relevance

This direction is relevant to unmanned vehicles, networked robots, autonomous surface vessels, and other cyber-physical systems where communication, sensing, or computation cannot be treated as continuously available resources.
